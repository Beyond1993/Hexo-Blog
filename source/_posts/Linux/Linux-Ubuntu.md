## 下载 youtube 视频

通过命令youtube-dl下载YouTube视频。
安装youtube-dl

复制代码代码如下:

$ sudo apt-get install youtube-dl

安装完毕后就可以用下面的格式下载TouTube视频：

复制代码代码如下:

$ youtube-dl url-to-video
