N Sum 也算是算法中的 Hello Word了

|问题|最优算法复杂度|
|:---|:-------------|
|2 sum| O(n)        |
|3 sum| O(n2)    |
|4 sum| O(n^2 *  logn) |
|...|...|
|n sum| O(n^(n-2) * logn)|

sum(int[][], int n, int target) {

  equals target 
  closest target
  smaller or equal
  bigger or equals
}
